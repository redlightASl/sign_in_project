# 以ESP32实现的控制及数据上传部分

使用ESP32与服务器进行通信，并通过SPI总线控制STM32的运行状态

## 功能

* 通过WiFi相关协议在内网收发数据
* 接收来自服务器的指令并发送给CPU（如低功耗待机、唤醒等）
* 收取服务器的指令并发送给STM32
* 获取SPI接口缓存的数据
* 将数据打包加密发送给WiFi模块


## 硬件特点

* ESP32-S模块最小系统
* 独立5V转3.3V供电
* 不支持独立烧录，需使用烧录器为ESP32烧录程序或使用STM32作为主机进行IAP烧录

## 使用方法

1. 从https://github.com/espressif/esp-idf获取ESP-IDF开发工具链

2. 按说明配置开发环境

3. 将本目录下文件放在项目文件夹中

4. 新建build目录并使用cmake工具进行配置

6. 使用开发工具进行编译、烧录

也可以使用VSCode及ESP32插件进行自动配置、烧录
